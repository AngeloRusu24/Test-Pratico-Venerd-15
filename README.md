# Centro Anchio

Sito vetrina responsive per un centro commerciale con spazi in affitto.
Il progetto è realizzato con HTML5 e CSS3 puro, senza framework esterni.

---

Tecnologie utilizzate
HTML5

Google Fonts
Features
Layout responsive
Hero section con overlay
Navbar sticky
Card animate hover
CSS Grid responsive
Palette elegante nero/oro
Struttura pulita e modulare
Responsive Design

Il sito utilizza:

grid-template-columns: repeat(auto-fit, minmax(...))

per adattare automaticamente le card a:

Desktop → 3 colonne
Tablet → 2 colonne
Mobile → 1 colonna
Font utilizzati
Playfair Display
Lato
